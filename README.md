# Library
##### React + GraphQL application

Starting the application:

```bash
git clone
cd library
npm install
cd server
npm install
cd ..
npm start
```

Scripts run React application on http://localhost:3000 and GraphQL on http://localhost:4000

If errors arise, run 
```bash
npm start
```

in server-directory and 
```bash
npm run react
```
in library-directory separately.
